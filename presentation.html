Real Example: t-test Blocking Scenario

STEP 1: User uploads cell viability data (n=30), right-skewed
STEP 2: Selects Independent t-test (Control vs Treatment)
STEP 3: Guardian Activates
✓ Sample Size: Adequate • ✓ Independence: OK • ✓ Variance: Homogeneous • ✗ Normality: VIOLATED (p < 0.001)
STEP 4: Guardian BLOCKS Test
🚫 TEST EXECUTION BLOCKED - Normality violated in both groups
Risk: Inflated Type I error rates, unreliable p-values
STEP 5: Evidence Provided
📊 Q-Q Plot: Severe deviation • 📊 Histogram: Right-skew • 📊 Shapiro-Wilk: p < 0.001
STEP 6: Alternatives Recommended
✅ Mann-Whitney U test (non-parametric) • ✅ Bootstrap CI • ✅ Permutation test
OUTCOME: User selects Mann-Whitney U → Guardian validates → Test runs → Valid results ✓
