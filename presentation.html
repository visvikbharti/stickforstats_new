How Guardian Works: Real Example

STEP 1: User Uploads Data

Cell viability measurements (n=30), highly right-skewed distribution

STEP 2: User Selects Independent t-test

Configuration: Control vs Treatment groups

STEP 3: Guardian Activates Automatically

✓ Sample Size: Adequate (n=30)
✓ Independence: No autocorrelation
✓ Variance: Homogeneous (Levene p = 0.12)
✗ Normality: VIOLATED (Shapiro-Wilk p < 0.001)
⚠ Outliers: 3 detected

STEP 4: Guardian BLOCKS Test Execution

🚫 TEST EXECUTION BLOCKED
Critical Violation: Normality assumption violated in both groups
This can lead to inflated Type I error rates and unreliable p-values

STEP 5: Guardian Provides Evidence

📊 Q-Q Plot: Severe deviation from diagonal
📊 Histogram: Clear right-skew visible
📊 Shapiro-Wilk: p < 0.001 (reject normality)

STEP 6: Guardian Recommends Alternatives

✅ Mann-Whitney U test (non-parametric)
✅ Bootstrap confidence intervals
✅ Permutation test

OUTCOME: Valid Results

User selects Mann-Whitney U → Guardian validates → Test runs → Scientifically sound results ✅
